ChatOps console

Query XORCISM and act on it conversationally — the same command set that drives the Slack / Teams integration. Try posture, digest, exposures, obligations, queue, then approve <id>. This console uses your session; Slack/Teams use the signed webhooks (see the docs).
Slack: add a slash command pointing at /api/chatops/slack/command and interactivity at /api/chatops/slack/interactive; set SLACK_SIGNING_SECRET. Approve/dismiss from chat needs CHATOPS_ALLOW_ACTIONS=1 and CHATOPS_TENANT.
