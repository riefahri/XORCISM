Agent Policy Firewall

A regulated pre-execution boundary for agent & automation actions — MCP calls, browser actions, cloud connectors, CLI tasks, SOAR playbooks and remediation bots. Every proposed action is scored for blast radius, matched against policies (allow / deny / require-approval), checked for replay and segregation-of-duties, and recorded with a tamper-evident signed receipt (SHA-256 hash chain) — before it runs. The governance layer agent-harness teams can't build from scratch. Complements the CROC loop (reactive) and the AI guardrails.
