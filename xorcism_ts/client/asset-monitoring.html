Asset Monitoring

Real-time uptime & health of your assets — HTTP / Ping / TCP / DNS / SSL / Server monitors with status (up / down / warning / paused), uptime %, response time, SSL expiry and up/down incidents. Create monitors here, or import them from CheckCle (the checkcle connector). Records land in MONITORINGCHECK / MONITORINGINCIDENT.
