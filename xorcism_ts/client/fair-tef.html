FAIR — Threat & Loss Event Frequency estimation

The frequency side of the FAIR ontology (complementing FAIR-MAM's loss magnitude). Estimate each factor as a PERT range (min / most-likely / max) and run a Monte Carlo simulation: LEF = TEF × Vulnerability, TEF = Contact Frequency × Probability of Action, Vulnerability = P(Threat Capability > Resistance Strength). Add a loss magnitude → Annualized Loss Expectancy (ALE) + a loss-exceedance curve. Save to write LEF/ALE back onto a risk-register entry.
