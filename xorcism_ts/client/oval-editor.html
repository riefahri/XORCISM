OVAL Definition editor

Author an OVAL definition and its criteria tree — reusing the OVAL tests and definitions already imported into XOVAL (140k+ tests, 35k+ definitions). The criterion leaves reference existing tests; extend_definition leaves reference existing definitions; AND/OR/ONE/XOR groups nest freely. Saving persists the definition relationally and generates an OVAL document in the chosen schema version (OVAL-Community 5.12.x / 5.11.2). Use ⬆ Import test content to load a full OVAL document so each criterion's test shows the object/states it checks (and can be inlined as self-contained output). New definitions get the oval:ai.xorcism:def:N namespace.
