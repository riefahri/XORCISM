Locataires (tenants)

Chaque locataire cloisonne ses données opérationnelles et ses utilisateurs. Le locataire Système (super-admin) voit tout.

Nom Utilisateurs Statut Actions
Sauvegarde des bases

Crée un instantané cohérent de toutes les bases dans DB_DIR/backups/<horodatage>/ (le serveur peut rester actif).

Corrélation CVE → actifs

Relie les CVE aux actifs par correspondance CPE (niveau produit) et crée les liens ASSETVULNERABILITY. Nécessite des CPE sur les actifs et l'import NVD des CPE affectés.

Notification à tous les utilisateurs

Diffuse une notification à tous les utilisateurs de votre locataire (le super-admin la diffuse à tous les locataires).

Rôles & permissions (RBAC / CRUD)

Cochez C(reate) R(ead) U(pdate) D(elete). L'enregistrement est automatique. Le rôle Admin a tous les droits.
