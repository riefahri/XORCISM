SIEM-lite — detection pipeline

Paste log events (JSON) and XORCISM runs them through your Sigma rule library plus a built-in baseline ruleset (Windows/Sysmon). Each match raises an ALERT — which feeds incidents and emits a LOOPEVENT, so the CROC loop and the agentic orchestrator pick it up and propose a response. Closes the detect→respond loop with no external SIEM.
